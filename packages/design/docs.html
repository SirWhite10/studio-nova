Connecting to the Cluster

Initialize the client using your project's connection string. Ensure you are using the latest stable release of the SDK for optimal performance.

typescript 

import { PrestigeClient } from '@prestigedb/client';

/* Initialize the prestigious client */
const client = new PrestigeClient({
  connectionString: process.env.PRESTIGE_URL,
  options: {
    poolSize: 10,
    timeoutMs: 5000
  }
});

await client.connect();
console.log('Successfully established connection to the luxury tier.');
